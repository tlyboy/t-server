# t-server

🌐 T-Server

## Install

```bash
git clone https://github.com/tlyboy/t-server.git
```

## Usage

Copy `.env.example` to `.env` and configure:

```sh
NITRO_JWT_SECRET=
NITRO_ENCRYPTION_KEY=

# MySQL database configuration
NITRO_MYSQL_HOST=
NITRO_MYSQL_PORT=
NITRO_MYSQL_USER=
NITRO_MYSQL_PASSWORD=
NITRO_MYSQL_DATABASE=
```

```bash
pnpm install
pnpm dev
pnpm build
```
